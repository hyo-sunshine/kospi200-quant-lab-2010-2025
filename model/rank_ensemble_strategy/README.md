# rank_ensemble_strategy — KOSPI200 압축 랭크앙상블 전략 (최종본)

> **전략 한 줄**: LightGBM 랭킹 모델 18개(2지평 × 9시드)의 앙상블로 KOSPI200 안에서
> 상대적으로 유망한 **상위 8종목을 동일가중 보유**, 10거래일마다 버퍼 규칙으로 교체.
>
> **검증 성과 (2013~2025 OOS, 편도 10bp)**
> | CAGR | 누적 | 롤링5년 중앙값 | 롤링5년 최악 | MDD | 50bp 비용 시 CAGR |
> |---|---|---|---|---|---|
> | **16.78%** | +645% | **+81.7%** | **+0.9%** | -57.5% | 12.7% |
>
> (KOSPI200 동기간: CAGR 5.8%, 누적 +111%)

## 전략 명세

1. **학습 타깃**: 각 날짜에 유니버스 전 종목의 미래 수익률 **횡단면 순위(0~1)**.
   절대 방향 예측이 아님 (방향 예측은 AUC 0.53으로 불가능함을 확인).
2. **모델 2지평 × 9시드 = 18개 LightGBM**
   - flow60 (느린): 60일 지평, 저속 피처 25개 — 재무 z 12 + 장기모멘텀 + 수급 20/60d + 시장
   - flow20 (빠른): 20일 지평, 전체 피처 39개 — 위 + 기술지표 + 단기모멘텀 + 수급 5d
   - 피처 핵심: 재무 건전성(부채비율·매출성장), 12-1 모멘텀, **외국인/기관/개인 수급
     (5/20/60일 누적)**, 환율(5/20일 변화), 시장 국면
3. **신호 합성**: 시드별 스코어 → 20일 평활 → 날짜별 rank pct → 9시드 랭크 평균
   → `signal = 0.3 × flow60 + 0.7 × flow20`
4. **포트폴리오**: 신호 상위 8종목 동일가중(12.5%), 10거래일마다 점검,
   보유 종목이 **32위(=8×4 버퍼) 밖**으로 밀렸을 때만 교체. 거래비용 편도 10bp 가정.
5. **재학습**: 연 1회 (반기 재학습은 실험 결과 이득 없음).

## 파이프라인

```
[검증]  train.py ──► model_scores/ ──► backtest.py  (성과·강건성)
[운용]  train_production.py ──► models/ ──► predict.py  (오늘의 8종목)
```

- **검증 경로**는 walk-forward(매년 과거로만 학습)라 미래정보가 0% — 성과 수치의 근거.
- **운용 경로**는 전 기간 학습 가중치 — 실제 매매 신호 생성용.

## 반드시 알아야 할 한계 (상세는 report/)

- **백테스트 ≠ 실거래**: 종가 체결 가정, 슬리피지 미반영.
- **MDD -57%**: 8종목 집중의 대가. 낙폭을 못 견디면 이 전략의 수익은 실현되지 않음.
- **약한 구간 존재**: 2017-2020은 지수 대비 열세 (절대수익 +29%). 롤링 5년 최악
  구간 +0.9% — "언제나 5년 2배"가 아니라 "장기 평균이 그 페이스".
- 모멘텀·수급은 공개 팩터 → 효과 약화 가능. **연 1회 재검증 필수.**
- 구성 파라미터(8종목, 버퍼 4배 등)는 과거 탐색의 결과 → 미래 성과는 표기보다
  보수적으로 기대할 것.
